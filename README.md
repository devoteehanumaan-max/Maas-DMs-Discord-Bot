# Mass DM Bot

A simple Discord bot to send mass DMs to server members.

## Commands:
- `!dmhelp` - Show all commands
- `!setmsg [message]` - Set DM message
- `!preview` - Preview message
- `!startdm` - Start sending DMs
- `!stopdm` - Stop sending
- `!dmstatus` - Check status
- `!testdm` - Test DM to yourself

## Setup:
1. Set `DISCORD_TOKEN` environment variable
2. Install: `pip install -r requirements.txt`
3. Run: `python app.py`
